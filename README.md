# 📚 Atividades - Estruturas de Dados e Análise de Algoritmos

Repositório dedicado à resolução de exercícios práticos e teóricos da disciplina de Estruturas de Dados e Análise de Algoritmos (Prof. Alexandre Montanha). O foco principal das implementações é o estudo prático da eficiência de sistemas e comportamento assintótico.

## 🎯 Objetivo do Repositório
Armazenar as implementações teóricas de algoritmos clássicos e consolidar a análise estrutural de laços de repetição, recursividade e condicionais para a determinação da complexidade de tempo no pior caso (Notação Big-O).
